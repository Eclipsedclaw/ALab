## This procedure is for Aramaki LAb ubuntu server's port forwarding with router TP-link Archer AX55

Use your VNC to get into the UI system If you don't have that yet, please check this: https://github.com/Eclipsedclaw/ALab/tree/main/remote_setup

After you get into the UI system, open the browser and enter:
```
http://tplinkwifi.net/
```
or
```
http://192.168.0.1
```
```
http://192.168.1.1
```

One of those will give you this page
<img width="1253" alt="Screen Shot 2022-10-27 at 3 18 34 PM" src="https://user-images.githubusercontent.com/37788723/198379323-10ee2a83-4b66-4585-9e91-65763ffeb04d.png">

contact me(zeng.jia@northeastern.edu) to get the local password.

After you entering the system, click on the advanced menu
<img width="1256" alt="Screen Shot 2022-10-27 at 3 20 07 PM" src="https://user-images.githubusercontent.com/37788723/198379605-e5402a6a-ce11-4965-81f9-fa793e6a9eb8.png">
Go to NAT forwarding -> port forwarding
<img width="1251" alt="Screen Shot 2022-10-27 at 3 20 42 PM" src="https://user-images.githubusercontent.com/37788723/198379699-6c4a3974-2d9b-4b02-8134-21d9ed952219.png">
hit add button and select all your options like below with the port number you wanted to forward input correctly
<img width="437" alt="Screen Shot 2022-10-27 at 3 22 35 PM" src="https://user-images.githubusercontent.com/37788723/198380040-9419c8aa-f113-4a9b-8d18-549d5414a2d5.png">

Hit SAVE and port forwarding is done!
